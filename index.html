v17.0 - AUTO UPGRADE EDITION 12 de Maio, 2026

🤖 SISTEMA DE AUTO-MELHORIAS (NOVO):

  • ✅ Painel AUTO Multifuncional: Nova aba dedicada com 8 módulos de automação independentes, cada um com toggle ON/OFF individual.
  • ✅ Auto-Compra de Estruturas: Compra automaticamente a estrutura mais cara que o jogador pode pagar em intervalos configuráveis (Lento/Normal/Rápido/Turbo).
  • ✅ Auto-Pesquisa: Pesquisa upgrades automaticamente assim que energia suficiente for acumulada, priorizando os de menor custo.
  • ✅ Auto-Cápsula: Abre cápsulas automaticamente com cooldown de 2s para não quebrar a lógica de recompensas.
  • ✅ Auto-Portal: Gira o Portal Dimensional automaticamente assim que o cooldown de 5 minutos for resetado.
  • ✅ Auto-Boost Máximo: Detecta quando não há boosts ativos e aplica um boost de emergência automaticamente.
  • ✅ Auto-Missão: Resgata missões completas automaticamente sem necessidade de interação manual.
  • ✅ Auto-Otimizador: Analisa o estado do jogo a cada 10s e sugere/aplica a melhor estratégia para maximizar EPS.
  • ✅ Auto-Prestige: Colapsa o sistema automaticamente quando o ganho de Matéria Escura atingir o threshold configurado (padrão: 10).

📊 PAINEL DE CONTROLE AUTO:

  • ✅ Dashboard de Status: Mostra quantos módulos estão ativos, ações realizadas e próximas ações agendadas.
  • ✅ Velocidades Configuráveis: Cada módulo tem sua própria velocidade (Lento 5s / Normal 2s / Rápido 1s / Turbo 0.5s).
  • ✅ Sistema de Desbloqueio: Módulos avançados (Auto-Prestige, Auto-Otimizador) requerem Matéria Escura para serem desbloqueados.
  • ✅ Indicador no Top Bar: Status do auto-sistema visível na barra de informações superior.
  • ✅ Log de Ações: Todas as ações automáticas são registradas no Log Cósmico com prefixo [AUTO].

⚙️ MELHORIAS TÉCNICAS v17.0:

  • ✅ Auto-Loop Separado: Sistema de automação roda em loop independente (500ms) para não interferir no game loop principal.
  • ✅ Persistência de Estado: Configurações AUTO salvas no localStorage com o save do jogo.
  • ✅ Proteção contra Spam: Cooldowns internos por módulo impedem ações em excesso que poderiam causar lag.
  • ✅ Aba Indicadora: Badge "NEW" na aba AUTO para fácil descoberta pelo jogador.

🚀 OTIMIZAÇÕES DE PERFORMANCE (ANTI-LAG):
